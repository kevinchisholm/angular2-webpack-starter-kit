# angular2-webpack-starter-kit

## A base template for any Angular2 Webpack project

## Dependencies

What you need to run this app:

* [Node.Js](https://nodejs.org)

## Instructions

### Setup

* Clone this repo
* Run this command: **npm install**
* Run this command: **npm start**
